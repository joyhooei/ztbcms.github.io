## 性能优化

### 软件层面的优化

- PHP
    - 开启OPCache
- Apache
    - 根据服务器的操作系统，进行MPM的参数调优
- CDN
    - 静态资源采用CDN,推荐使用七牛，利用其镜像功能能快速把资源目录`d/`放置到CDN上
- MySQL 参数调优
